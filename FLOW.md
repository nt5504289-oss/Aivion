# 🔄 Data Flow Explanation

1. User enters input in frontend
2. JavaScript captures input
3. Fetch request sent to Flask backend
4. Backend processes request
5. Sends prompt to Groq API
6. AI generates response
7. Backend returns JSON
8. Frontend displays output

## Key Concepts

- JSON → data exchange format
- API → communication layer
- Async JS → handles delays
- Flask → backend server
